## 10. Responsive Behavior

### Desktop (≥1280px)

- All widgets visible
- Side decorations active
- Custom cursor enabled
- Full parallax effects

### Tablet (768px - 1279px)

- Some widgets hidden
- Reduced parallax
- System cursor
- 2-column layouts

### Mobile (<768px)

- All decorative widgets hidden
- Single column layout
- Simplified animations
- Touch-friendly spacing

```
Desktop                    Tablet                     Mobile
┌────┬────────┬────┐      ┌──────────────┐           ┌──────┐
```
