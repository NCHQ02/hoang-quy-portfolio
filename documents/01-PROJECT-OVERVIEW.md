### Core Visual Principles

1. **Spatial UI** - Elements float in 3D space with parallax effects
2. **Dark Mode First** - Deep dark background (#0a0a0a) for immersion
3. **Glassmorphism** - Frosted glass effects with backdrop blur
4. **Meta-UI** - UI within UI (browser headers, dock, panels)
5. **Physics-based Animation** - Natural, weighted movements
